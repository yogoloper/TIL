# 비밀번호 암호화 하기 설명
``` bash
$ npm install bcryptjs --sav
```

## 비밀번호 저장하는 방법
1. 원본 그대로 -> 최악
2. 비밀번호 암호화키로 암호화 (양방향)  
  -> 복호화가 가능해서 암호화 키가 노출되면 위험함
3. SHA256등 Hash로 암호화 (단방향)  
  -> 레인보우 테이블을 사용해서 해킹시도를 하면 위험함
4. 솔트(salt), salt + 비밀번호를 Hash로 암호화
  -> bcrypt에서 사용하는 방식
