# NestJS란?
nodejs의 프레임워크의 한 종류이며,  
express를 기본값으로하여 HTTP 서버 프레임워크를 사용한다.

## NestJS의 철학
angular에서 영감을 받아서 만들어진 NestJS는 개발자와 팀이 고도로 테스트 가능하고  
확장가능하며, 느슨하게 결합되고, 유지 관리가 쉬운 애플리케이션을 만들수 있는  
즉시 사용가능한 아키텍쳐를 제공한다.
