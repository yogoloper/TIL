# `Antd CSS Framework`
- https://ant.design/

## `CSS Framework 종류`
- Material UI
- React Bootstrap
- Semantic UI
- Ant Design
- Materialize

## `ant design 설치`
- client 에서 antd 설치
``` bash
$ npm install antd --save
```

## `ant design 사용`
- 스타일 시트 추가
``` javascript
// client/index.js

import 'antd/dist/antd.css'; // or 'antd/dist/antd.less'
```
