# `Create React App`
## `Babel`
- 최신 자바스크립트 문법을 지원하지 않는 브라우저들을 위해서  
  최신 자바스크립트 문법을 구형 브라우저에서도 돌수 있게 변환시켜 줌
## `Webpack`
- 웹사이트의 사이즈가 커져서 자바스크립트의 프레임워크나 라이브러리들을 사용하다보니  
  Webpack을 이용해서 번들화해줌

## `React 설치`
- 리액트를 설치할 디렉토리로 이동 후 아래 명령어 입력
``` bash
$ npx create-react-app .
```
