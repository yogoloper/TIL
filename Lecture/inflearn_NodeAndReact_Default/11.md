# `로그인 with Bcrypt`
## '로그인 기능 구현'
### `이메일 존재 여부 확인 및 비밀번호 일치여부 확인`
``` javascript
// index.js

app.post('/login', (req, res) => {
  User.findOne({ email: req.body.email }, (err, userInfo) => {
    // 요청된 이메일 있는지 확인
    if (!userInfo) {
      return res.json({
        loginSuccess: false,
        message: '이메일이 존재 하지 않습니다.'
      });
    }
    // 이메일 존재하면 비밀번호 맞는지 확인
    userInfo.comparePassword(req.body.password, (err, isMatch) {
      if (!isMatch)
        return res.json({
          loginSuccess: false,
          message: '비밀번호가 틀렸습니다.'
        });
    });
  });
});
```
