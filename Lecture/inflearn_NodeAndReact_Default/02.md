## `ExpressJS`

## `Node 환경설정`
### `nvm, node, npm 설치`
#### Homebrew 설치
``` bash
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
#### nvm 설치
``` bash
$ brew install nvm
```
#### 환경 변수 설정
``` bash
$ mkdir ~/.nvm
$ vim ~/.zshrc

# .zshrc 파일에 아래 내용 추가
export NVM_DIR=~/.nvm
source $(brew --prefix nvm)/nvm.sh

$ source ~/.zshrc
# 버전확인
$ nvm -v
```
#### node 설치
``` bash
$ nvm install 16
```
#### node 설치 확인
``` bash
$ nvm ls
$ node -v
$ npm -v
```

### `project 시작`
- package.json 파일에 아래 내용 추가
``` bash
  "scripts": {
    "start": "node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
```
- project 구동
``` bash
$ npm run start

> nodeandreact_default@1.0.0 start
> node index.js

Example app listening on port 5000
```
