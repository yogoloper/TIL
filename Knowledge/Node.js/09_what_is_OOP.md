# OOP란?
사물을 객체회 하여 프로그래밍 하는 기법

## 특징
### 추상화
객체의 모델링을 할때 공통적인 속성과, 기능을 정의
### 캡슐화
접근 제어 지시자를 통해 정보 은닉
### 상속
상위 클래스/인터페이스를 상속받아 코드 재사용성 증가
### 다형성
상속과 관련 있는 성질로 상속 받아 생성된 객체가 여러가지 형태로 재구성 되는 것
