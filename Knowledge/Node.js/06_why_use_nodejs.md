# Node.js 사용 이유?
1. javascript로 서버 프로그래밍을 할 수 있다.
2. 별도의 서버 프로그램이 없이도 http 서버 라이브러리를 포함하고 있어서 웹서버 동작이 가능하다.
3. 브라우저 외의 다른 용도의 서버로도 사용이 가능하다.
4. npm을 통해 다양한 모듈을 사용 할 수 있다.

위의 나열한 이유를 제외하고서,  
Node.js를 만든 Ryan Dahi는 javascript가 I/O를 차단하지 않는 것이 가장 큰 장점이라고 말했다.  
Non-blocking I/O는 요청을 한 번에 받고 처리속도가 빠른 것부터 처리 후 오래 걸리는 것은 나중에 작업하게 된다.  
이는 node.js는 요청이 많은 채팅, SNS 등의 서비스를 만들기에 적합하다.
